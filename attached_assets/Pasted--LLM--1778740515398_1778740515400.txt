## 1) System Pipeline

```text id="tb10pb"
Dataset
```

## 2) Evaluation Methodology

شرح:

* كيف يعمل الـ Judge
* كيف تُحسب Human averages
* كيف يُحسب Spearman
* معنى Overrating / Underrating

---

## 3) Error Taxonomy

شرح جميع الـ flags:

| Flag           | Description                        |
| -------------- | ---------------------------------- |
| PROMPT_LEAKAGE | Model repeated instructions        |
| HALLUCINATION  | Fabricated information             |
| OVER_VERBOSE   | Excessively long misleading answer |
| FACTUAL_ERROR  | Medically or legally incorrect     |
| PARTIAL_ANSWER | Incomplete response                |
| OFF_TOPIC      | Irrelevant answer                  |

---
